Sobre o Portal de Dados Abertos

Este portal foi desenvolvido para facilitar a consulta e análise de dados públicos, como notas fiscais, fornecedores e órgãos governamentais.

A aplicação utiliza Angular 19 com Angular Material no frontend e uma API em Java Spring Boot no backend, garantindo performance, responsividade e boa experiência de uso.

O objetivo é tornar a visualização de informações públicas acessível, organizada e fácil de navegar — contribuindo com transparência e pesquisa.
